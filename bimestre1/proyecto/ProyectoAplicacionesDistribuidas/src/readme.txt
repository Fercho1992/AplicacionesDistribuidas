Chat Cliente-Servidor usando UDP

1.Se inicia al Servidor.
2.Se inicia el Cliente.
3.Se ingresa por teclado el nombre de usuario (UserName), del lado del cliente.
4.Se ingresa por teclado el nombre de usuario (UserName), del lado del Servidor.
5.El nombre ingresado por teclado por el lado del cliente es enviado al Servidor.
6.El nombre ingresado por teclado por el lado del servidor es enviado al Cliente.
7.Por el lado del servidor se pide que escribamos un mensaje.
8.Por el lado del cliente se comienza a enviar mensaje los cuales van a se vistos 
por el lado del servidor.
9.Por el lado del servidor se van a poder los mensajes enviados del lado del cliente.
10.De aqui en adelante la conversacion es fluida y el unico que la puede terminar es 
el cliente, el servidor no termina el simplimente espera hasta una proximo mensaje
por parte del cliente.
